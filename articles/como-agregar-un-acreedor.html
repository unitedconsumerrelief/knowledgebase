How to Add a Creditor

Step-by-step guide to adding creditors to the debt settlement system

Overview

This guide will walk you through the process of adding creditors to the debt settlement system. Follow these steps carefully to ensure accurate data entry.

Step-by-Step Instructions

1

Press the Creditors option

Navigate to the main menu and select the 'Creditors' option to access the creditor management section.

2

Enter the creditor name

In the creditor name field, enter the full legal name of the creditor as it appears on the credit report or billing statement.

3

Select the account type

Choose the appropriate account type from the dropdown menu. Common types include credit cards, personal loans, medical bills, and student loans.

4

Add the current debt amount

Enter the current outstanding balance on the account. Use the most recent statement or credit report for accuracy.

5

Select debt owner

Choose who owns the debt. Typically, you will select 'Applicant' for individual accounts. For joint accounts, select the appropriate owner.

6

Press Save

Click the 'Save' button to add the creditor to the system. Verify all information is correct before saving.

Important Notes

Accuracy is crucial

Ensure all creditor information is entered accurately. Incorrect information may affect the debt settlement process and client outcomes.

Troubleshooting

Creditor not found in system

If the creditor name doesn't appear in the search results, you can add it manually by entering the full legal name.

Invalid account type

If the account type you need is not in the dropdown, contact your supervisor for guidance on how to proceed.

Save button not working

Ensure all required fields are filled out completely. The save button will only be active when all mandatory information is provided.
